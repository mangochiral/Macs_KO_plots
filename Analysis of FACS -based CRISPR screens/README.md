Analysis of FACS-based CRISPR screens usinf SLICeVLP platform in primary humna macrophages

1. Screen for CD80 surface expression
   - Library: Transcription factor and immunomofulator library
   - Input files:
      - sgRNA list 'tf_library_sgRNAs.csv' in metadata
     - Fastq files submitted to GEO
   - This screen was performed in cells from four human donors, labeled D1, D2, D3, D4
  
2. Screen for TNF production in response to LPS stimulation
     - Library: Myeloid-specific library
     - Input files:
          - sgRNA list 'Myeloid_lib_sgRNAs.csv' in metadata
          - Fastq files submitted to GEO
      - This screen was perfoemed in cells from two human donors, labeled D1, D2
