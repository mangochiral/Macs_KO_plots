
Analysis of bulkRNA seq data from primary human macrophages treated with Cas9EDV carrying either AAVS1 or TNFAIP3 sgRNA.

Bulk RNA seq for GSEA

  Groups : NTD, Cas9EDV(gAAVs1), Cas9EDV(gTNFAIP3(A20)), labeled NTD, AAVS1, A20
  Fastq files submitted to GEO
  This bulkRNA seq was performed in cells from four human donors, labeled D1, D2, D3, D4

  Gene set enrichment analysis (GSEA) following differential expression analysis with DESeq2, using MSigDB KEGG(left) and Hallmark(right) pathway gene sets. Genes included in the analysis were filtered based on adjusted p-value (Padj < 0.05) and Log2 fold change (|Log2FoldChange| > 1).
